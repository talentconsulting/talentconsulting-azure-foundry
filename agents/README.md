# Agents

Store agent manifests, instructions, tool access, safety constraints, and release notes here.

Recommended structure:

```text
agents/
└── agent-name/
    ├── manifest.yaml
    ├── instructions.md
    ├── tools.yaml
    ├── guardrails.md
    ├── evaluations.md
    └── release-notes.md
```

Do not commit secrets, live credentials, personal data, or production conversation transcripts unless explicitly approved and anonymised.
